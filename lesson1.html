Перевод чисел из десятичной системы счисления и обратно.

Что такое система счисления?

Система счисления - это символический метод записи чисел, представление чисел с помощью письменных знаков.
Проще говоря, система счисления - это совокупность знаков для записи чисел. Существуют различные системы счисления.
Та, которой мы с вами пользуемся, называется десятичной, т.к. в ней используется 10 знаков (от 0 до 9 включительно).

Основание десятичной системы равно 10

Для обозначения цифр в системах счисления с основанием больше 10 будут использоваться другие символы. Так, цифра 10 будет обозначаться A, цифра 11 - B
и так далее. Для шестнадцатеричной системы счисления этот ряд продолжится вплоть до 15, которая будет обозначаться F.

Перевод чисел в систему с основанием N

Для перевода чисел из десятичной системы счисления в систему счисления с основанием N существует алгоритм:

  1. Делим исходное число на N
  2. Записываем остаток от деления
  3. Повторяем всё с шага 1 до тех пор, пока целая часть от деления не станет равна 0
  4. Записываем все остатки в обратном порядке

Пользуясь данным алгоритмом, переведём число 1110 в двоичную систему:

  1. Разделим 11 на 2, получим целую часть 5 и остаток 1
  2. Разделим 5 на 2, получим целую часть 2 и остаток 1
  3. Разделим 2 на 2, получим целую часть 1 и остаток 0
  4. Разделим 1 на 2, получим целую часть 0 и остаток 1
  5. Полученную строку из остатков (1101) запишем в обратном порядке
  6. Получим 10112 - число 1110

Перевод чисел в десятичную систему

Для перевода чисел в десятичную систему из системы с основанием N также существует алгоритм:

  1. Расставим степени над цифрами исходного числа справа налево начиная с 0
  2. Умножим каждую цифру исходного числа на N в степени, которую проставили в предыдущем шаге
  3. Сложим все полученные значения

Пользуясь данным алгоритмом, переведём число 10112 в десятичную систему:

  1. Расставим степени над каждой цифрой числа 10112 (13021110)
  2. Умножим каждую цифру на N в степени, которую над ней проставили (1 * 23 = 8, 0 * 22 = 0, 1 * 21 = 2, 1 * 20 = 1)
  3. Сложим все полученные значения (8 + 0 + 2 + 1 = 11)
  4. Получим 1110 - число 10112
